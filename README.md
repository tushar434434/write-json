# ✍️ Write JSON in Rust

A simple Rust project that demonstrates how to create and write JSON data to a file using Rust. This project showcases serialization and file handling with `serde` and `serde_json`.

## 🚀 Features

* Create structured data using Rust structs
* Serialize Rust objects into JSON
* Write JSON data to a file
* Simple and beginner-friendly implementation

## 🛠️ Built With

* **Rust**
* **Serde**
* **Serde JSON**

## ⚡ Getting Started

### Clone the Repository

```bash
git clone https://github.com/tushar434434/write-json.git
cd write-json
```

### Build the Project

```bash
cargo build
```

### Run the Project

```bash
cargo run
```
